## What are Tutorials?

Tutorials are **learning-oriented** guides that take the reader through a series of steps to complete a project or learn a skill. They are:

- **Beginner-friendly** - Assume minimal prior knowledge
- **Step-by-step** - Clear, sequential instructions
- **Goal-oriented** - Achieve something concrete by the end
- **Supportive** - Explain "why" along the way

## Tutorial Guidelines

### Structure

1. **Prerequisites** - What's needed before starting
2. **Learning Objectives** - What you'll learn
3. **Steps** - Sequential instructions with explanations
4. **Summary** - What was accomplished
5. **Next Steps** - Where to go from here

### Writing Tips

- Write for beginners
- Explain the "why"
- Include screenshots where helpful
- Provide complete code examples
- Anticipate common issues
