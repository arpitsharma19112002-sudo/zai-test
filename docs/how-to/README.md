# How-To Guides

Task-oriented guides for accomplishing specific goals.

---

## What are How-To Guides?

How-To guides are **problem-oriented** documents that help you accomplish a specific task. They are:

- **Goal-focused** - Solve a specific problem
- **Practical** - Direct, actionable steps
- **Flexible** - Assume you know the basics
- **Adaptable** - Multiple approaches where relevant

## How to Use These Guides

1. **Identify your goal** - What are you trying to accomplish?
2. **Find the relevant guide** - Browse the categories above
3. **Follow the steps** - Each guide has clear instructions
4. **Adapt as needed** - Modify steps for your context
