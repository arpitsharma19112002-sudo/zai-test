# Project Roadmap

This document outlines the high-level vision, milestones, and planned features for zai-test.

## Vision

To provide a well-structured, scalable repository template that enables teams to start projects with best practices in documentation, AI prompt management, and development workflows from day one.

---

## Milestones

### v0.1.0 - Foundation (Current) ✅

**Status:** Complete

- [x] Initial project structure
- [x] Core documentation framework
- [x] GitHub templates (issues, PRs)
- [x] AI prompt folder organization
- [x] Basic CI/CD workflow templates

---

### v0.3.0 - AI Integration

**Target:** Q3 2026
